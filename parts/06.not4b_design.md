4-Bit Not Design
================

```
Inputs: in[4];
Outputs: out[4];

Parts:
  not1 NOT,
  not2 NOT,
  not3 NOT,
  not4 NOT;

Wires:
  in[1] -> not1.in,
  in[2] -> not2.in,
  in[3] -> not3.in,
  in[4] -> not4.in,

  not1.out -> out[1],
  not2.out -> out[2],
  not3.out -> out[3],
  not4.out -> out[4];
```
